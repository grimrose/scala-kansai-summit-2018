scala-kansai-summit-2018
=============================

## Akkaを分散トレーシングで見てみよう

### requirements

* docker
* docker-compose
* sbt
* pipenv
* [kt3k/saku](https://github.com/kt3k/saku)

### usage

[commands](saku.md)

### setup

```bash
$ cp .env.sample .env
```

```bash
$ pipenv install
```

### demo 

```bash
$ saku stage
```

```bash
$ saku start
```
